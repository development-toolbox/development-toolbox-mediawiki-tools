üí° Quick Tips & Essential Info

Key things to know before starting your migration

üöÄ Preview Before Migration

Use the Content Previewer to see exactly how your content will convert:

python migration/content_previewer.py --page "PageName"

‚ö° Automation Levels

100% Automated - No manual work needed
85% Automated - Minor adjustments needed
Manual Required - Significant manual work

üñºÔ∏è Images = Manual Process

Images must be downloaded from Azure DevOps and uploaded to MediaWiki manually. Plan extra time for image migration.

üìä Complex Tables

Tables with merged cells or complex formatting may need manual adjustment. Test with sample pages first.

üéØ Best Practices

Follow these guidelines for smooth migration

üìù Before Migration
  • Simplify complex HTML in Azure DevOps where possible
  • Standardize image references for easier replacement
  • Document custom formatting that needs special handling
  • Create a backup of your Azure DevOps content
üîÑ During Migration
  • Test with a small batch of pages first
  • Migrate in batches to catch issues early
  • Keep track of manual changes needed for consistency
  • Verify each page after conversion
‚úÖ After Migration
  • Create MediaWiki templates for common formatting
  • Train users on MediaWiki syntax differences
  • Set up style guidelines for future content
  • Monitor and fix any broken links
üîß Quality Checklist
  • Headers display with correct hierarchy
  • Code blocks have syntax highlighting
  • Tables display properly
  • All links work correctly
  • Images are uploaded and referenced

    return True

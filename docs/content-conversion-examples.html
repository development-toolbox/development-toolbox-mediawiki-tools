Image references and attachments

üìù Azure DevOps Markdown

Input

‚ö†Ô∏è Manual Conversion Required

Steps Required
# 1. Download images from Azure DevOps
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "https://dev.azure.com/org/project/_apis/wiki/wikis/wiki/attachments?path=architecture.png" \
  -o architecture.png

# 2. Upload to MediaWiki (use upload interface or API)

# 3. Update references in MediaWiki syntax:
[[File:Architecture.png|Architecture Diagram]]
[[File:Workflow.jpg|thumb|Process Flow]]
[[File:Screenshot.png|500px|UI Screenshot]]

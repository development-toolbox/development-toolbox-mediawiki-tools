üöÄ Getting Started

Step-by-step guide to migrate your wiki in 6 phases

‚ö†Ô∏è Prerequisites: Ensure you have Python 3.7+, access to both Azure DevOps and MediaWiki, and proper API permissions.

Phase 1: Environment Setup

Install dependencies and configure your environment.

Installation
# Clone the repository
git clone https://github.com/development-toolbox/development-toolbox-mediawiki-tools.git
cd development-toolbox-mediawiki-tools

# Install dependencies
pip install -r migration/requirements.txt

# Set up environment variables
cp .env.example .env
# Edit .env with your credentials

Phase 2: Analysis

Analyze your Azure DevOps wiki to understand migration complexity.

Run Analysis
python migration/migration_planner.py
üìÑ Output: Creates migration_analysis_report.md with detailed recommendations.

Phase 3: Environment Preparation

Set up your MediaWiki test environment and configure access.

Docker Setup (Optional)
# Start local MediaWiki for testing
docker-compose up -d

# Access at http://localhost:8080

Phase 4: Content Preview

Preview how your content will convert before migration.

Preview Content
# Preview specific pages
python migration/content_previewer.py --page "Home"

# Generate sample conversions
python migration/content_previewer.py --sample-pages 5

Phase 5: Template Setup

Import necessary MediaWiki templates for consistent formatting.

Import Templates
python templates/mediawiki_import.py --import templates/

Phase 6: Migration Execution

Execute the migration with monitoring and validation.

Run Migration
# Start with a test batch
python migration/azure_devops_migrator.py --wiki "ProjectDocs" --batch-size 10

# Full migration
python migration/azure_devops_migrator.py --wiki "ProjectDocs"
‚úÖ Success! Your wiki has been migrated. Review the conversion quality checklist in the documentation.

üìù Conversion Examples

See how different content types convert from Azure DevOps to MediaWiki

Headers & Formatting

Azure DevOps (Markdown)
# Main Title
## Section Title
**Bold text** and *italic text*
MediaWiki (Converted)
= Main Title =
== Section Title ==
'''Bold text''' and ''italic text''

Code Blocks

Azure DevOps (Markdown)
```python
def hello_world():
    print("Hello, World!")
```
MediaWiki (Converted)
<syntaxhighlight lang="python">
def hello_world():
    print("Hello, World!")
</syntaxhighlight>

Tables

Azure DevOps (Markdown)
| Name | Role | Status |
|------|------|--------|
| John | Dev  | Active |
MediaWiki (Converted)
{| class="wikitable"
|-
! Name !! Role !! Status
|-
| John || Dev || Active
|}

üõ†Ô∏è Troubleshooting

Common issues and solutions

üö® Authentication Issues

Problem: Unable to connect to Azure DevOps or MediaWiki

Solutions:

  • Verify Personal Access Token has proper permissions
  • Check MediaWiki API credentials
  • Ensure network connectivity
  • Validate .env file configuration

üìÑ Content Conversion Errors

Problem: Content not converting properly

Solutions:

  • Use content previewer to identify issues
  • Check for unsupported HTML elements
  • Validate markdown syntax in source
  • Review complex table structures

üñºÔ∏è Image Migration Issues

Problem: Images not displaying correctly

Solutions:

  • Images require manual download/upload
  • Update image references after upload
  • Check MediaWiki file upload permissions
  • Verify image file formats are supported

‚ö° Performance Issues

Problem: Migration taking too long

Solutions:

  • Use batch processing for large wikis
  • Check API rate limits
  • Optimize network connectivity
  • Consider running during off-peak hours
üí° Pro Tip: Always test with a small batch of pages first, and keep backups of your original content before migration.

üìã Prerequisites
  • Python 3.8 or higher
  • Azure DevOps access token
  • MediaWiki admin credentials
  • Git (for cloning the repository)
1
Install Dependencies
Set up Python environment and required packages
pip install -r requirements.txt
2
Configure Connection
Set up Azure DevOps and MediaWiki credentials
python setup_config.py --azure-token YOUR_TOKEN --mediawiki-url YOUR_URL
3
Analyze Content
Run migration analysis to understand your content
python migration_planner.py --wiki "YourWikiName"
4
Preview Migration
Preview how your content will convert
python content_previewer.py --page "YourPageName"
5
Execute Migration
Run the full migration process
python azure_devops_migrator.py --wiki "YourWikiName"

üîó Code Samples

Basic Usage
Advanced Configuration
Custom Scripts
from azure_devops_migrator import WikiMigrator

# Initialize migrator
migrator = WikiMigrator(
    azure_url="https://dev.azure.com/yourorg",
    mediawiki_url="https://wiki.yourorg.com"
)

# Run migration
migrator.migrate_all()

from azure_devops_migrator import WikiMigrator, Config

# Advanced configuration
config = Config(
    batch_size=50,
    enable_templates=True,
    convert_links=True,
    preserve_history=True
)

migrator = WikiMigrator(config=config)
migrator.migrate_with_progress_callback(callback=my_progress_handler)

from azure_devops_migrator import CustomProcessor

class MyCustomProcessor(CustomProcessor):
    def process_content(self, content):
        # Custom content transformation
        content = self.convert_custom_macros(content)
        return super().process_content(content)

# Use custom processor
migrator = WikiMigrator(processor=MyCustomProcessor())
